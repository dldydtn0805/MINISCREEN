## 개요

- 크롬 확장 프로그램입니다.
- 화면 위에 보조 화면을 띄울 수 있습니다.
- MENIFEST V3를 사용합니다.
- IFRAME으로 화면을 가져옵니다.

## 설명

- 상단 검색 바에 주소를 입력하면 그 주소로 이동하고 검색어를 입력하면 구글 검색 결과를 가져옵니다.
- 헤더를 드래그 해서 창을 이동할 수 있습니다
- 북마크를 등록할 수 있습니다
- 북마크를 끄고 켤수 있습니다

## 설치방법

- 주소 창에 `chrome://extensions/` 로 이동합니다 
- 우상단 `개발자 모드` 를 켭니다
- 좌상단 `압축 해제된 확장프로그램을 로드합니다` 클릭
- 폴더 선택
